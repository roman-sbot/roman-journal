# roman-journal 📚
Jurnal pribadi tempat merawat pikiran, ide, cerita harian, dan catatan perjalanan. Menghadirkan pengalaman membaca yang tenang, bebas gangguan, dan interaktif dengan sentuhan antarmuka modern yang simpel, rapi, serta dilengkapi fitur indikator bacaan, pemutar musik melayang, dan pembaca suara otomatis.
